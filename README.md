# OshoWorld.com Audio Downloader

A Python-based scraper and downloader for OshoWorld.com Hindi audio discourses.

## Features
- Fetches all Hindi discourse series
- Handles nested sub-series (e.g. Geeta Darshan)
- Reliable episode pagination
- Selective download via regex or list
- Progress display with per-series ETA
- Resume-safe (skips existing files)

## Requirements
- Python 3.10+
- requests
- beautifulsoup4

## Usage
```bash
` python downloader.py `
